Standup Assistant

This web application is designed as a supplemental tool to be used in conjunction with our Google Assistant action in order to record, review, and maintain daily standups for your teams.

It was created using Spring Boot, Bootstrap CSS, and jQuery. The application is currently hosted on Amazon Web Services. You are able to manage and view daily standups across multiple teams from individual team members.
